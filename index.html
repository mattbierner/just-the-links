Customization?

Add a Whitelisted Word

It is *highly recommended* that you add your name to the list of whitelisted words. But you can add other whitelisted words easily too.

1. Open the plugin options.

2. Add the new word. Here I've added Blot're. Words are comma separated. You can whitelist multi word phrases too. Basic regular expressions also work.

Toggle

Disable or enable the plugin through the toggle option.

This effect only applies to the current page and is lost if you reload.

Permanently Disable For Page/Site

Open the plugin menu.

Disable For Page turns the plugin off for just the current page. Disable For Site turns the plugin off for the entire website.

Or

1. Open the plugin options.

2. In the list of Excluded Sites, add the site you wish to disable the plugin for. To disable for just one page, add its url to the list. To disable for an entire site, include the domain name plus /* at the end.
